# BITshare

BITshare is a web-based BIT Yanhe course workspace built from the current `frontend` and `backend`.

## Structure

- `frontend`: Vue + Vite web application
- `backend`: Node.js API service

## Local development

### Backend

```powershell
cd backend
npm install
npm start
```

### Frontend

```powershell
cd frontend
npm install
npm run dev
```
